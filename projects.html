01 / Runtime

ReAgent

打开仓库

一个本地优先的研究工作流运行时与 CLI，用来承载长生命周期任务、记忆和可复用输出。

技术栈

Node.js、TypeScript、Prisma、CLI runtime、workspace memory。

当前状态

目前更偏基础设施与运行时能力建设，适合展示系统设计与编排思路。

  • 强调状态可见，不把流程藏进黑盒。
  • 输出结果可持续保存，结束后仍能被再次利用。
  • 把 research、memory 和 runtime 放进统一运行时。
02 / Product

TravelAgent

打开仓库

一个围绕旅行规划场景构建的多步骤工作台，把自由输入转成更接近真实可执行的行程方案。

技术栈

Java 21、Spring Boot、Spring AI、Vue 3、Vite、SQLite、Amap。

当前状态

目前更偏产品化表达，适合展示从模型能力到界面工作流的闭环。

  • 把不同专家能力组织成路由式工作流。
  • 通过 Amap 做 grounded enrichment，减少纯文本规划的漂浮感。
  • 界面和执行链路都更接近真实产品，而不是技术演示。
