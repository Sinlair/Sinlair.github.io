Selection Logic

What makes a repository worth surfacing here.

Technical bar

  • It shows a real system or workflow, not only a code fragment
  • There is enough structure to inspect how it was built
  • The repo points to a direction I plan to keep pushing

Presentation bar

  • The project can be explained in product terms, not only stack terms
  • The README and public framing make the repository legible
  • The work is useful as a signal of taste and direction
