01 / Runtime

ReAgent

Open repository
作用

本地优先的研究工作流运行时与 CLI，用来承载长生命周期任务、记忆和可复用输出。

技术栈

Node.js、TypeScript、Prisma、CLI runtime、workspace memory。

项目亮点
  • 强调可见状态，不把流程藏进黑盒
  • 输出结果可持久保存，结束后仍可复用
  • 把 research 与 memory 放进统一运行时
当前状态

更偏基础设施与运行时能力建设，适合展示系统设计思路。

02 / Product

TravelAgent

Open repository
作用

围绕旅行规划场景的多步骤工作台，把自由输入转成更接近真实可执行的行程方案。

技术栈

Java 21、Spring Boot、Spring AI、Vue 3、Vite、SQLite、Amap。

项目亮点
  • 把不同专家能力做成路由式工作流
  • 通过 Amap 做 grounded enrichment
  • 界面和执行链路都更接近真实应用
当前状态

更偏产品化表达，适合展示从模型能力到界面工作流的闭环。
